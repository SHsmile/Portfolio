Welcome to the portfolio page of Xiao Wei. This portfolio contains some interesting projects about video games, music and some artwork of Xiao Wei.

Darkness-RPG game

Created a complete role-playing game demo based on Unity3D. Players can switch the control of character and wizard, manipulating the wizard to trigger the trap to beat monsters. And collected materials to walk to the end of the game.

Mountain of Faith-A Shooting Game

Independently designed a 2D shooting game. Players shoot bullets to beat enemy, and upgrade by randomly catching the tools dropped from enemies. Multiple levels and bosses were set with different attack models.

download

Link Game-A casual game

Independently developed a 2D casual game, in which players eliminating all the blocks with same pattern in the shortest time win the game.

Cuisine Assistant

Designed and developed an application to help people learn cooking skills through voice teaching, freeing people from frequent watching book reference during the cooking. The system issues voice to direct users in the cooking steps and accepts user feedback. It spots specific keywords to perform repeat, skip and backward operations.
